# School System Project

Este proyecto implementa un sistema escolar básico en Java con dos clases principales: Student y Course. Permite gestionar estudiantes, cursos, inscripciones y evaluaciones.

## Contenido

### Student
La clase Student incluye:
- Nombre y apellido
- Matrícula
- Calificación
- Año escolar
- Métodos para imprimir nombre completo, validar aprobación y avanzar de año
- Tres constructores diferentes para crear objetos de manera flexible

### Course
La clase Course incluye:
- Nombre del curso
- Nombre del profesor
- Año del curso
- Lista de estudiantes inscritos

Métodos implementados:
- Inscribir estudiantes
- Desinscribir estudiantes
- Contar número de inscritos
- Obtener la mejor calificación
- Sobrecarga del método enroll para aceptar arreglos de estudiantes

Funciones adicionales:
- Calcular el promedio del curso
- Generar ranking de estudiantes por calificación
- Indicar si cada estudiante está por encima o por debajo del promedio

### Main
La clase Main permite probar todas las funcionalidades del sistema mediante la creación de objetos Student y Course.

## Requerimientos
- Java 8 o superior
- Eclipse o cualquier entorno compatible

## Ejecución
Compilar y ejecutar la clase Main para ver los resultados en consola.
